[アプリケーションの概要]

    タイトル: P2Pホワイトボードアプリ

    機能:

        手書き入力・入力した内容の共有

            手書き入力された図形の移動・削除

            手書き入力の文字色の設定変更

        付箋貼り付け・貼り付けた付箋の共有

            貼り付けられた付箋の移動・削除

            付箋に設定された文字列の修正

            付箋の地の色の設定変更

            付箋の文字色の設定変更

        ホワイトボード自体への描画内容の変更

            方眼紙のようなマス目を表示

            真っ白な描画への変更

[使用技術]

    JavaScriptフレームワーク: Vite（Reactは使用しない）

    フロントエンド: HTML, CSS, JavaScript

    P2Pライブラリ: Peer.js

    状態管理: Nano Stores

[ロジック]

P2P接続のロジックやタイミングは次のリポジトリと同一としてください。

    https://github.com/yzkn/CloudflarePages-Vite-P2PEditor

Peerを識別するID生成には、以下の関数を利用する

    // 指定された文字セットでIDを生成する関数
    const ALPHABET = '123456789abcdefghijkmnopqrstuvwxyz';
    function generateShortId(): string {
        let result = '';
        for (let i = 0; i < 4; i++) {
            result += ALPHABET.charAt(Math.floor(Math.random() * ALPHABET.length));
        }
        return result;
    }
