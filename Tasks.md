P2Pホワイトボードアプリ 機能実装タスクリスト

# 1. P2P接続・基盤機能

状態	機能内容	テスト結果・挙動確認
✅	4文字の短縮ID生成	ページ起動時、1-9, a-zからなる4文字のID（例: 3x9j）が自動生成されることを確認。
✅	URLハッシュによるルーム共有	生成されたIDが index.html#ID としてURLに反映され、そのURLを共有することで他者が接続可能。
✅	Peer.jsによる接続・同期	「Join」ボタンから他者のIDを入力して接続。接続成功時に「Connected」のインジケーターが緑色に点灯。
✅	状態の全体同期 (SYNC_ALL)	後から参加したユーザーに対して、既存の描画内容や付箋がすべて即座に反映される。
✅	Firefox接続エラー対策	STUNサーバーの設定により、WebRTCのICE失敗エラーを回避し接続性を向上。

# 2. 手書き入力（鉛筆ツール）

状態	機能内容	テスト結果・挙動確認
✅	自由曲線描画	カラーピッカーで選択した色を用いて、マウスドラッグで滑らかな線が描ける。
✅	線のリアルタイム同期	描画中、他のユーザーの画面にもリアルタイムで線が伸びていく。
✅	線の移動機能	描画済みの線をクリックしてドラッグすることで、座標を移動できる。
✅	描画のちらつき防止	移動中に render をスキップするロジックにより、ドラッグが途切れない。

# 3. 付箋（Sticky Note）機能

状態	機能内容	テスト結果・挙動確認
✅	付箋の作成	付箋ツールを選択してクリックすると、現在のカラーピッカーの色を背景色として新規付箋が作成される。
✅	付箋の移動	付箋をドラッグしてボード上の好きな位置に配置できる。
✅	付箋のリサイズ	右下のハンドルをドラッグして、内容に合わせて大きさを自由に変更できる。
✅	ダブルクリック編集	付箋をダブルクリックすると編集モーダルが開き、テキストを修正できる。
✅	背景色・文字色の変更	モーダル内のカラーピッカーで、付箋ごとの背景色・文字色を個別に設定可能。
✅	フォントサイズ変更	数値入力により、付箋内の文字サイズを 8px〜100px の範囲で変更可能。
✅	付箋の削除	編集モーダル内の「Delete」ボタンから、個別の付箋を削除できる。

# 4. ホワイトボード操作・設定
状態	機能内容	テスト結果・挙動確認
✅	背景の切り替え (Grid/Plain)	ツールバーのボタンで「方眼紙（Grid）」と「真っ白（Plain）」を切り替え。他者の画面にも同期される。
✅	IDコピー機能	ID横のコピーボタンをクリックすると、4文字のIDのみがクリップボードにコピーされる。

# 5. UI/UX・アクセシビリティ
状態	機能内容	テスト結果・挙動確認
✅	SyncLN風ヘッダー	モダンなフローティング・ツールバーを実装。レスポンシブ対応によりモバイルでも操作可能。
✅	Lucideアイコンの表示	import 方式によるアイコン描画。接続時の再描画エラーも refreshIcons 関数で解消済み。
✅	Tabキーによるフォーカス制御	モーダル表示時に背後の要素にフォーカスが逃げない（Focus Trap）。付箋をTabで選択してEnterで編集可能。
✅	初期フォーカス	ページ読み込み時、またはモーダル表示時に適切な入力項目（PeerID入力やテキスト編集）に自動でフォーカスが当たる。
